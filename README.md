# Stats

Projet de visualisation football en Python.

## Structure

- `src/` : scripts Python reutilisables (theme, utilitaires, etc.)
- `notebooks/` : notebooks d'analyse et d'exploration
- `figures/` : exports de graphiques et visuels
- `requirements.txt` : dependances Python

## Installation

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Lancement

Ouvrir les notebooks dans `notebooks/` depuis VS Code/Jupyter.
